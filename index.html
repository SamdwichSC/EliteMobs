EliteMobs WebApp

Welcome to the Elite Mobs webapp.

This app is made for both the absolute beginner and those that just want an easier way of creating anything custom.
Choose Interactive Mode if you want as much help as possible! This mode is not recommended for experienced users as the excessive handholding will most likely be more annoying than useful. Choose Select Forms if you're an experienced user. If you choose multiple forms at the same time, they'll automatically connect with eachother where possible.
